# Receipts

Receipts preserve observation and review history.

Use receipts to answer:

- When did we observe this?
- Where did it come from?
- What method was used?
- What rights status applied?
- What public display status applied?
- What record, claim, or snapshot did this receipt support?
- Was the item stored, blocked, summarized, or flagged for review?

Receipts should follow `schemas/receipt.schema.json`.
