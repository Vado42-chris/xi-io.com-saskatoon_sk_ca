# Data

This directory is for structured snapshots, derived data, receipts, and indexes.

Do not use this directory as a raw scraper dump.

Recommended subdirectories:

```txt
data/snapshots/
data/derived/
data/receipts/
data/indexes/
```

## Metadata that every stored data item should preserve

- Source ID
- Source URL or archive reference
- Access timestamp
- Rights status
- Licence notes
- Parser or method used
- Hash where practical
- Review status
- Public display status
- Link to related records and claims
