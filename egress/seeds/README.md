# Seeds Egress

Seeds output is generated from linked civic-memory clusters.

Do not create a Seed from a single unsupported input.

Minimum cluster before a Seed:

- Place record where relevant
- Timeline entries where relevant
- Source records
- Claim records
- Receipt records
- Archive references where relevant
- Media references as metadata, not copied articles
- Asset references, not unauthorized copies
- Neighbourhood or ward links
- Conflict or uncertainty notes
- Rights status
- Public-safe summary

Possible outputs:

- Local history article
- Neighbourhood profile
- This-week civic brief
- Then-and-now article
- Dashboard card
- Council backgrounder
- Research request
