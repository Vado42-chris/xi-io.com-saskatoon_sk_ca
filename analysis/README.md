# Analysis

This directory is for derived work, not primary source storage.

Analysis outputs may include:

- Neighbourhood summaries
- Time-series notes
- Source-quality notes
- Missing-data notes
- Conflict and correction logs
- Research questions
- Pattern and anomaly notes

Every analysis file should link back to source records, claims, and receipts.

Avoid unsupported conclusions. Keep facts, assumptions, risks, and recommendations distinct.
