# Structural Conditions

This lane stores analysis notes and derived baseline context for civic pressure systems such as food insecurity, poverty, housing, accessibility, mobility, cost of living, service gaps, and intervention outcomes.

## Rule

Structural condition data is context. It is not proof about individuals, households, private addresses, or block-level safety.

## Recommended uses

- Baseline profiles
- Food security analysis
- Housing pressure analysis
- Service-gap analysis
- Mobility and accessibility analysis
- Monthly and yearly trend context
- Seeds article backgrounders
- Dashboard context panels

## Required warnings

Use warning notes when indicators are:

- provincial proxy values
- national proxy values
- mixed-granularity values
- stale values
- missing denominator values
- not comparable across years
- not comparable across geographies

## Egress rule

Do not turn structural poverty or food insecurity into neighbourhood danger scores. Use it to identify unmet needs, access barriers, service gaps, and policy conditions.
